# QuickMeet Swift Complete

Native SwiftUI foundation for the QuickMeet social app.

## Features
- Auth entry
- Onboarding
- Discover / swipe-like flow
- Profile detail
- Match success modal
- Messages list
- Chat
- Profile
- Edit profile
- Local persistence via UserDefaults
- Theme tokens and reusable components

## Setup
This repo includes `project.yml` for XcodeGen.

```bash
brew install xcodegen
xcodegen generate
open QuickMeet.xcodeproj
```
