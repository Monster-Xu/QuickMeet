# QuickMeet

A SwiftUI MVP for a fast-match social app.

## Included
- Onboarding
- Discover
- Profile Detail
- Match Success Sheet
- Messages
- Chat
- Profile
- Edit Profile

## Setup
This repo includes a `project.yml` for XcodeGen.

```bash
brew install xcodegen
xcodegen generate
open QuickMeet.xcodeproj
```
