# Screen Specs

## Onboarding
Goal: complete first-time setup quickly.
Main action: enter app.

## Discover
Goal: make profile browsing and matching feel premium and fast.
Must emphasize:
- large image-first card
- name and age
- online state
- tags
- pass / like / super-like actions

## Profile Detail
Goal: support conversion from interest to action.
Main actions:
- like
- super-like

## Match Modal
Goal: create emotional feedback and push user into chat.
Main action:
- chat now

## Messages
Goal: make conversations easy to scan.
Must show:
- avatar
- name
- online state
- last message
- unread badge

## Chat
Goal: reduce chat friction.
Must show:
- clear message hierarchy
- realistic input area
- icebreaker suggestions

## Profile
Goal: help user understand their account state.
Must show:
- avatar
- display name
- completion state
- settings and edit actions

## Edit Profile
Goal: save user profile changes and reflect them in app state.
